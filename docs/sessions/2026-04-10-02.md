4. **Pages built** (all with mock data):

### Theme system

- CSS custom properties (`--primary`, `--sidebar-bg`, etc.)
- HSL-based color derivation (dark, light, muted, alpha variants)
- Sidebar background auto-adapts to chosen color
- All buttons, charts, badges, focus rings use theme vars
- Login page also respects saved theme
