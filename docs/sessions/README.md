# Session Logs

Track what was accomplished in each working session. This helps maintain context across conversations.

## How to Use

After each session, create a log file named `YYYY-MM-DD-NN.md` (NN = session number that day).

## Template

```markdown
# Session YYYY-MM-DD-NN

**Stage**: Which stage was worked on
**Duration**: Approximate
**Goal**: What we set out to do

## Done
- [ ] Task completed
- [ ] Another task completed

## Decisions Made
- Decision and reasoning

## Blockers / Issues
- Any problems encountered

## Next Session
- What to pick up next
```
