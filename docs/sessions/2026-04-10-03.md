# Session 2026-04-10-03: Dashboard Backend + Real Data + Deployment

## What was done

### Backend API endpoints

1. **Auth module** (`modules/auth/`):
   - Admin model with bcrypt password hashing
   - JWT login + me endpoints
   - @fastify/jwt plugin with authenticate decorator
   - Auto-seed default admin on empty DB

2. **Orders** (`modules/orders/`):
   - List, detail, status update via Odoo sale.order
   - State mapping: Odoo (draft/sale/done/cancel) ↔ Dashboard (pending/ready/dispatched/cancelled)

3. **Chats** (`modules/chats/`):
   - Aggregated chat list from MongoDB messages
   - Message history, send manual message

4. **Customers** (`modules/customers/`):
   - List/detail from MongoDB User model

5. **Products** (`modules/products/`):
   - List with search, detail from Odoo

6. **Stats** (`modules/stats/`):
   - Summary: orders, revenue, customers
   - Agent: bot vs manual percentages
   - Charts: daily sales, weekday sales, hourly activity, orders by status, top 10 products, top 10 customers

7. **WhatsApp** (`modules/whatsapp/`):
   - List/create/delete instances via Evolution API

### Frontend connected to real API

All pages now fetch from `api.leofarmacia.com`:
- Orders, Products: from Odoo
- Chats, Customers: from MongoDB
- WhatsApp: from Evolution API
- Stats/Charts: aggregated from Odoo + MongoDB
- No mock data remaining
